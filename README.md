Find the Killer

A text-focused, dynamic clicker and deduction game built in Android Studio. Players must identify a hidden killer among survivors by strategically interviewing NPCs, powered by a randomized role-generation engine.

⚠️ **CONTENT WARNING:** This project contains mature dialogue and themes (NSFW text). The primary focus of this repository is to showcase game architecture, dynamic state management, and algorithmic randomness.

---

🛠️ Technical Highlights & Mechanics

*Randomized Role Engine (RNG):** Implemented an algorithmic system that dynamically assigns the roles of "Killer" and "Survivors" at the start of each session, ensuring unique playthroughs.
*State-Driven Dialogue System:** Programmed a conditional text interview mechanic. The system parses distinct dialogue lines depending on whether the NPC is a survivor or the generated killer, providing clues based on player input.
*Clicker Progression Logic:** Developed a responsive clicker mechanic to manage in-game progression and pacing.
*Resourceful Asset Pipeline:** Utilized generative AI (Bing Image Creator) to efficiently produce character portraits and background assets, allowing full focus on rapid prototype deployment and core programming logic.

🧰 Tech Stack
*IDE:** Android Studio
*Language:** Java

🚀 How to Play
1. Engage with the core clicker loops to progress.
2. Interview NPCs using the text interface.
3. Analyze the randomized dialogue clues to deduce the killer's identity!
